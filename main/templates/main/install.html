Installation instructions:

  • Download 4 Organizer from this URL
  • Run file "Setup4-Organizer.exe"
    • If you have Windows 8/8.1/10 you may see this message:
  • Press on "More info" text.
  • Then, press "Run anyway".
  • After short installation, on the Desktop will appear "4-Organizer" shortcut. Run it.
    • You may again see "Windows protected your PC" press "More info" and then "Run anyway".

    4 Organizer is installed! Now you will see it's main window.
