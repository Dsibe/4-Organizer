FAQ

Is 'Forever' plan really forever?

Yes, it's forever-forever!

Can 4-Organizer delete my valuable files?

No, that is not possible. 4-Organizer will never delete your files.

Where can I find my license key?

License key can be found in profile page.

What is "Machines manager"?

"Machines manager" is a special tool which helps you to view, ban and unban machines connected to unique license key. With it, you can ban certain machines from using your license.

Someone stole my license key! What do I need to do?

Please contact us as soon as possible. We will help you to create a new key.

My antivirus says that 4-Organizer is a trojan / virus, is that true?

4-Organizer is not a trojan or a virus. Sometimes antiviruses "think" that 4-Organizer is a trojan. It happens because 4-Organizer sorts your files. While sorting, 4-Organizer moves and sometimes renames files. For antivirus programs, this behaviour is suspicous. So, they "think" that 4-Organizer is a trojan.
You can always check it by yourself. For example, upload 4-Organizer's executable to VirusTotal and read the detailed analysis. You will be able to see every single action that program does.

Do I have to enter license data every time?

Thankfully, no. You only need to enter license data once.

Can I exclude certain folders from sorting?

Yes, it is possible! When starting scan, deselect folders where sorting is undesired.

How 4-Organizer solves name conflicts?

Suppose you have file "example.txt" in folder "Downloads" and "example.txt" in folder "Desktop". 4-Organizer will rename one of the files. So, after sorting, there will be "example.txt" and "(1) example.txt".

Is it possible that "Remove empty folders" tool will delete my files?

No, that is not possible. This toll will only remove folders which are absolutely empty.

Will I receive updates?

You will receive updates as long as your license is valid.

I've forgot my license key, what do I have to do?

Please contact us.

I've got some technical problem, what do I need to do?

Please contact us

What will happen after the license expires?

4-Organizer will deactivate. If you'll buy another license, you can activate it again.
